# 課題２

## データセットの作成

### 1.priceの標準化
priceを扱いやすい大きさにするため全てのデータをその中の最大値で割ることで0~1の間に丸めた。
### 2.入力データの標準化
入力ベクトルからそれぞれの次元の平均を引き、標準偏差で割ることで平均０、分散１に落とし込み、のちに作成するニューラルネットで扱いやすい形にする。
### 3.データの分割
今回はscikit-learnを用いて訓練データ、検証用データ、テスト用データをそれぞれ6:2:2に分割する
